# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to Semantic Versioning.

## [0.1.0.0] - 2026-07-14

### Added
- Initial launch of Previso Replenishment Autopilot per SME italiane.
- CSV Column Mapping Wizard interface to dynamically ingest custom ERP layouts.
- Dynamic Replenishment Dashboard displaying safety stock proposals and draft PO reviews.
- SKU Master Data configuration grid for modifying unit costs, lead times, and supplier details.
- Purchase Order generator compiling A4 invoice PDFs using jsPDF.
- Email dispatch modal overlay simulating supplier notifications.
- Vitest testing environment with 7 unit tests verifying MIN-MAX thresholds, MOQ rounding, duplicate prevention, SKU updates, and supplier creation.
